# Сервис коротких ссылок — Laravel + React

Сервис для создания коротких ссылок с возможностью отслеживания статистики переходов и поддержкой коммерческих ссылок с показом рекламы.
Backend — Laravel 12, frontend — React с TypeScript и Vite.

## Описание проекта

Проект реализует сервис коротких ссылок. Пользователи могут создавать короткие ссылки с кастомными именами, устанавливать срок действия, отслеживать статистику переходов. Поддерживаются коммерческие ссылки с показом рекламного баннера перед редиректом.

---

## Технологии

- Backend: Laravel 12 (PHP 8.2)
- База данных: MySQL 8+
- Frontend: React 18, TypeScript, Vite
- Стили: Tailwind CSS

## Установка и запуск

Команды для запуска без docker:

- Backend (Laravel):
cd backend
composer install
cp .env.example .env
# настройте .env (база данных)
php artisan key:generate
php artisan migrate
php artisan serve

- Frontend (React):
cd frontend
npm install
npm run dev

- Доступ после запуска:

Backend: http://localhost:8000
Frontend: http://localhost:5173

## Использование

- Зарегистрируйтесь или войдите в систему
- На главной странице введите длинную ссылку
- При желании укажите:
    Свой вариант короткой ссылки
    Срок действия
    Отметьте как коммерческую ссылку (с рекламой)
- Получите короткую ссылку
- Отслеживайте статистику переходов по ссылке "Статистика" рядом с каждой ссылкой

## API

Публичные маршруты:
- POST /api/register — регистрация
- POST /api/login — вход
- GET /api/{code} — редирект по короткой ссылке

Защищенные маршруты (требуется авторизация)
- GET /api/links — список ссылок пользователя
- POST /api/links — создание новой ссылки
- DELETE /api/links/{id} — удаление ссылки
- GET /api/stats/{slug} — статистика переходов
- GET /api/stats/unique/14days — уникальные посетители за 14 дней для всех ссылок
